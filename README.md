# GitHub Pages 배포용 정적 변환 파일

## 파일 구성
- `index.html` : 대시보드 메인
- `login.html` : 데모 로그인 화면
- `join.html` : 데모 회원가입 화면

## 주의
이 버전은 GitHub Pages에서 실행되도록 JSP/Servlet/MySQL 기능을 제거하고
브라우저 `localStorage` 기반 데모 동작으로 바꾼 버전입니다.

## GitHub Pages 배포 방법
1. 새 GitHub 저장소 생성
2. 이 폴더 안 파일들을 업로드
3. GitHub 저장소의 `Settings` → `Pages`
4. Branch를 `main`, Folder를 `/ (root)`로 설정
5. 저장 후 생성된 주소 접속
